# Q8 Solution: Dependabot Configuration

## Objective
Configure Dependabot to automatically check for security updates in a Python project.

## Steps Taken

1.  **Project Setup**:
    -   Created a Python project structure.
    -   Added `requirements.txt` with dependencies: `fastapi`, `requests`, `pandas`.

2.  **Dependabot Configuration**:
    -   Created `.github/dependabot.yml`.
    -   Configured to check `pip` (Python) dependencies.
    -   Schedule set to `weekly`.
    -   Commit message prefix set to `deps`.

3.  **Deployment**:
    -   Initialized a Git repository.
    -   Pushed to `https://github.com/priyanshuyadav705/tds-ga2-q8`.
